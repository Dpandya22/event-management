# Event Management System 🎉

A simple Django project to manage events, bookings, and schedules.

## 🔧 Features
- Admin login
- Event creation & editing
- Attendee registration
- Razorpay Integration

## 🛠️ Tech Used
- Python, Django
- HTML, CSS, Bootstrap 
- SQL
